# 7. Business logica – HES (werknaam)

Status: Definitieve annex
Doel: Formuleren van regels, aannames en optimalisatiegedrag.

Keten:
woningcontext -> verbruik -> productie -> warmtepomp -> EV -> batterij -> netbalans -> financieel -> optimalisatie

Kernregels:
- optimalisatie alleen op vrijgegeven componenten
- contractcontext beinvloedt batterij- en EV-waarde
- resultaten moeten zowel technisch als financieel verklaarbaar zijn
- multi-objective mogelijk: financieel, onafhankelijkheid, gebalanceerd
