# 5. Gebruikersflow – HES (werknaam)

Status: Definitieve annex
Doel: Beschrijven van de klantreis van onboarding tot advies.

Hoofdflow:
Landing -> Woningprofiel -> Energieprofiel -> Bestaande systemen -> Optimalisatievoorkeuren -> Doelselectie -> Analyse -> Advies -> Scenariovergelijking -> Detailanalyse -> Actie

UX-regels:
- begin eenvoudig, onthul complexiteit progressief
- toon steeds waarom informatie gevraagd wordt
- geef expliciete feedback bij aannames, warnings en blokkades
- advies moet direct begrijpelijk zijn en daarna verdiepend
